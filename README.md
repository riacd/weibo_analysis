
## 前端界面运行方法
### 环境配置
>1. 安装所需Python包：pip3 install streamlit
### 加载前端界面
2. 使用streamlit指令运行前端界面（注意当前所处目录应该在weibo_analysis/目录下）：
    streamlit run main_page.py
3. 此时用浏览器打开终端提示的网址即可看到对应页面
